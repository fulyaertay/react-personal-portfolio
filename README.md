## ðŸ”‹ Installation

**1. Clone the Repository**

```
  git clone https://github.com/fulyaertay/react-personal-portfolio
```

**2. Move into the working directory**

```
  cd personal-portfolio
```

**3. Install Dependencies**

```
  npm install
```


**4. Run the App**

```
  npm start
```
